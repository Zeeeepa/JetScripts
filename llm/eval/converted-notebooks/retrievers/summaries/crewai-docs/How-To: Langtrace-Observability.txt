This guide provides step-by-step instructions on how to monitor cost, latency, and performance of CrewAI Agents using an external observability tool. It covers setting up the tool, integrating it with CrewAI, and utilizing its features for deep visibility into agent performance. The process involves signing up for the tool, creating a project, installing the SDK, importing and initializing the tool in your script, and then importing CrewAI modules. The guide also highlights various features of the tool, including LLM token and cost tracking, trace graphs for execution steps, dataset curation with manual annotation, prompt versioning and management, prompt playground with model comparisons, and testing and evaluations.
